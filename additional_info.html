Virtual table top

Virtual tabletops (VTT) or tabletop simulators are video game programs that are designed to allow users to recreate existing games or create their own games to play with others online. The game typically provided a game engine with numerous pre-made tabletop game assets like dice, tokens, and cards, and allows players to create or import their own.

The game may have limited scripting ability so that some of the tabletop game's functions can be automated, but otherwise, players are expected to know the rules and play them out as given, simply using the virtual tabletop as a game board. This can create copyright concerns since art assets from the published game get distributed freely through user mods. Some tabletop game developers have embraced these programs by offering their games as official downloadable content for the simulator program.

Roll20

Roll20 is a website consisting of a set of tools for playing tabletop role-playing games, also referred to as a virtual tabletop, which can be used as an aid to playing in person or remotely online. Roll20 is a browser-based suite of tools that allows users to create and play tabletop role playing games. It is organized into individual game sessions, which users can create or join.

These game sessions include various features of typical tabletop RPGs, including dynamic character sheets, automated dice rolling, shared maps with basic character and enemy tokens, and triggered sound effects, as well as a character creation tool for certain licensed game systems. The interface also includes integrated text chat, voice chat, and video chat, as well as Google Hangouts integration.

We recommend using Roll20 for firstime users of VTTs. It is free and in our experience the easest to use of curently available free VTTs with integrated features from thrd party developers like Tabletop audio. Roll20 also has a subscription option that adds additional features to improve the game experience, but for majority of ganes us ut bi requered.

Additional VTTs

While Roll20 is one of the most popular VTTs, there are other options. Each VTT has its own pros and cons, some are easer to get into, others have more feature and dedicated community working on add-ons to enhance the game experience. Bellow is the list of VTTs that we use:

  • Foundrt VTT - one time purchase software with dedicated community that works hard on add-ons to imrpove the game experience. Downside of this VTT is that hosting a game requers either renting of cloudserver or port forwarding
  • Astral Tabletop - free to use VTT that has feature similar to roll 20, features like dynamic ligting are free and easy to use. Downside of this VTT is that it can be buggy for players somtimes.
  • Fantasy Grounds - is a paid software that contains various build in features for enhanced game experience. We recommend this software to people that already have experience with VTTs, and know what they experience they want for their gaming group. Downside of this software is that for every game system and module you have to purchase an additional add-on.

For some of your game, whether you are playing theatre of the mind or a more tactical grid-based game, you will need a map. Map can be hand drawn or made in various software. Maps provide a visual aid for players of a place or location where their characters find themselves in. Usually there are three types of maps that we use in our games:

  • Tactical map – usually used in combat to give players visual information where objects, enemies, obstacles, and allies are at any point in time. These maps usually cover a couple of rooms, a building, or a dungeon.
  • Town map – gives a visual layout of a base location that could be any type of settlement (city, village, space station, bunker, ship) with markers of key locations that players will visit during the game.
  • Regional map – sometimes also called world maps gives an overview to players where the game takes place. This type of map is used to give players an overview of key locations in the game, feeling of distance and travel, as well as landmarks that give substance to the game world.

Maps can contain as many or little details that are needed. Tactical maps are best to have enough details to give players the feeling of where the most important objects are on the map. For a city map, it is enough to have simple city blocks and pointers to key locations. As a general rule, it is best to build the regional map from the inside out starting from the base location. This gives GMs an easier way to plan the next adventure, and players have an experience of exploration that provides that the world is big, mysterious and there are places to be charted.
